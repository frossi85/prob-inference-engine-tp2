# prob-inference-engine-tp2

## Uso

Para poder correr el modelo es necesario tener instalado ProbLog:

Python 2: ```sudo pip install problog```
Python 3: ```sudo pip3 install problog```

El modelo se puede ejecutar con el comando

```problog model.pl```
